# 🎯 Demo Day Pro Tips

## Before You Walk In (30 min)
1. Run pre-demo checklist
2. Load app in incognito
3. Test receipt upload with real image
4. Time yourself: 5:00-5:30
5. Deep breath, you've got this

## During Demo

### If Gemini Is Slow (>2 sec)
- Don't panic — say: "API şu anda biraz yoğun, bir saniye bekliyoruz..."
- Let it load, it will work
- Use the delay to explain what's happening

### If Feature Fails
- Have screenshot backup ready
- Say: "Canlı demoda bazen gecikme olabiliyor, error handling'imiz bunu yakalıyor"
- Show error boundary UI
- Move on confidently

### If Judges Ask Technical Questions
- **"LangGraph nedir?"** → "5 ajanı orkestreleyen bir sistem. Her ajan uzmanlaşmış bir alanı yönetir."
- **"Neden Gemini?"** → "Türkçe dil desteği, Vision API (fiş okuma), düşük maliyet, yüksek doğruluk."
- **"Nasıl scale edersiniz?"** → "Prisma + PostgreSQL yatay skalaya uygun. Docker containerize."
- **"Security?"** → "Clerk authentication, environment variable'lar, Prisma type-safe sorgular."

### Magic Moments (Highlight These)
- Health Score gauge animating ✨
- Anomaly detection red box appearing 🎯
- Receipt scanner extracting data 📷
- Chat response coming back with agent name 🤖
- Investment allocation bars showing 📊

## Tone
- Confident but humble
- Turkish fluently
- Professional but warm
- Excited about the product
- Respectful of judges' time

## Energy
- Stand (don't sit)
- Move around screen
- Make gestures
- Make eye contact
- Smile genuinely

## If Time Runs Out
- Skip to Investment + Chat
- Show innovation (receipt scanner)
- Show depth (multi-agent)
- Leave them impressed

## GOLD MOMENT
When judges are silent/thinking after demo = they're impressed.
Let it breathe. Say: "Sorular?" Wait 3 seconds.

**YOU'VE GOT THIS. 🏆**
